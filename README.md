# 🤖 Telegram Master Bot

بوت تيليجرام احترافي مدعوم بالذكاء الاصطناعي (Gemini) لإدارة القنوات ونشر إشارات الفوركس.

---

## ✨ المزايا

| الميزة | الوصف |
|--------|-------|
| 🧠 ذكاء اصطناعي | تحليل وصياغة البوستات عبر Gemini 1.5 Flash |
| 📡 إشارات الفوركس | معالج خطوة بخطوة لإنشاء ونشر الإشارات |
| 📢 إدارة القنوات | ربط قنوات متعددة والنشر التلقائي فيها |
| 📅 جدولة المنشورات | نشر مجدول عبر node-cron |
| 🏦 MongoDB | حفظ الإشارات والقنوات والمنشورات |
| ☀️ تقرير صباحي | تقرير يومي تلقائي عن السوق كل صباح |

---

## 🚀 التثبيت والتشغيل

### 1. المتطلبات
- Node.js 18+
- حساب Telegram Bot (من [@BotFather](https://t.me/BotFather))
- مفتاح Gemini API (من [Google AI Studio](https://aistudio.google.com))
- قاعدة بيانات MongoDB (اختياري — [MongoDB Atlas](https://cloud.mongodb.com) مجاناً)

### 2. الإعداد

```bash
# انسخ ملف الإعدادات
cp .env.example .env

# عدّل المتغيرات في .env
notepad .env

# ثبّت الحزم
npm install

# شغّل البوت
npm start
```

### 3. ملف `.env`

```env
BOT_TOKEN=توكن_البوت_من_BotFather
GEMINI_API_KEY=مفتاح_Gemini
MONGODB_URI=رابط_قاعدة_البيانات         # اختياري
ADMIN_IDS=123456789,987654321            # معرفاتك على تيليجرام
```

> **الحصول على معرفك:** أرسل رسالة لـ [@userinfobot](https://t.me/userinfobot)

## 📖 الأوامر

### عامة
| الأمر | الوصف |
|-------|-------|
| `/start` | الشاشة الرئيسية |
| `/help` | قائمة الأوامر |

### ذكاء اصطناعي
| الأمر | الوصف |
|-------|-------|
| `/analyze [نص]` | تحليل وصياغة بوست فوركس |
| `/ask [سؤال]` | أسئلة عامة للذكاء الاصطناعي |

### إشارات الفوركس (للمشرفين)
| الأمر | الوصف |
|-------|-------|
| `/signal` | إنشاء إشارة جديدة (خطوة بخطوة) |
| `/signals` | آخر 5 إشارات |
| `/publish [رقم]` | نشر إشارة في القنوات |

### إدارة القنوات (للمشرفين)
| الأمر | الوصف |
|-------|-------|
| `/addchannel [-100xxx]` | إضافة قناة (البوت يجب أن يكون مديراً فيها) |
| `/channels` | عرض القنوات المتصلة |
